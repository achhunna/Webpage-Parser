Webpage Parser

You can use Wepage Parser to scrape websites which have common URL with multiple pages. For e.g. careershifters.org lists stories in numbered pages with URL below.

  • Enter your desired URL below along with page range you want to scrape
  • Indicate HTML Attr (attribute) you want to parse, e.g. div, p, section; for class, prefix with "."; for id, prefix with "#"
  • Keep Output Format to "Raw" for website other than Career Shifter. Single page scraping can also be done by leaving the First Page as blank

The code for the tool is on my GitHub. I appreciate your feedback.

Enter URL:
http://www.careershifters.org/success-stories?field_type_of_work_tid=All&field_to_tid=All&page=
First Page:
0
Leave empty for single page
Second Page:
19
HTML Attr:
.node__title
Prefix "." for class, "#" for id
Output Format:
Parse
